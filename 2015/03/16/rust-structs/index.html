Rust - 结构体

Mar 16 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

结构体（Structs）分为三种：

  • 元组结构体（Tuple structs），即命名元组（named tuples）
  • 经典结构体（classic C structs）
  • Unit 结构体，不含成员域，常用于泛型

13.1 可见性

结构体包含额外的可见性约束，其成员域可公有可私有（默认私有）。一个结构体的可见性仅仅在它对外可被访问时有用，这有助于信息封装。
