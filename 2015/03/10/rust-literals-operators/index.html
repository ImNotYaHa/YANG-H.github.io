Rust - 字面值与运算符

Mar 10 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

整数1，浮点数1.2，字符a，字符串abc，布尔值true以及 unit 类型()都可以用字面形式直接书写。

整数可以通过添加0x，0o或0b前缀来表示成十六进制、八进制或二进制。

数字写法中间添加下划线可以提高可读性，如1_000等价于1000，0.000_001等价于0.000001。

我们需要告诉编译器每个字面常量的具体类型。当前我们使用u32后缀来声明一个无符号 32 位整数，使用i32后缀来声明一个有符号 32 位整数。下一章将详细介绍类型系统，字面常量的类型标注方法详见这一章。

常用运算符及其优先级与类 C 语言近似。

fn main() {
    // 整数相加
    println!("1 + 2 = {}", 1u32 + 2);

    // 整数相减
    println!("1 - 2 = {}", 1i32 - 2);
    // TODO ^ 尝试把 `1i32` 改成 `1u32` 来体会为何类型如此重要

    // 布尔运算
    println!("true AND false is {}", true && false);
    println!("true OR false is {}", true || false);
    println!("NOT true is {}", !true);

    // 位运算
    println!("0011 AND 0101 is {:04b}", 0b0011u32 & 0b0101);
    println!("0011 OR 0101 is {:04b}", 0b0011u32 | 0b0101);
    println!("0011 XOR 0101 is {:04b}", 0b0011u32 ^ 0b0101);
    println!("1 << 5 is {}", 1u32 << 5);
    println!("0x80 >> 2 is 0x{:x}", 0x80u32 >> 2);

    // 使用下划线提高可读性
    println!("One million is written as {}", 1_000_000u32);
}
