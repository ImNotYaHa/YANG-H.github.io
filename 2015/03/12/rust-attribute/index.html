Rust - 属性（Attributes）

Mar 12 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

属性是用于描述模块、箱子或其它项目的元数据。属性可被用于：

  • 代码的条件编译
  • 设置箱子的名字、版本以及类别（是可执行程序还是库）
  • 禁用特定警告
  • 启用某些编译器特性（宏、全局导出(global import)等等）
  • 链接到非 Rust 库（foreign library）
  • 将函数标记为单元测试
  • 将函数标记为 benchmark 的一部分

将属性用在整个箱子上时，对应的语法是 #![crate_attribute]，当用在模块或者其它项目上时，对应的语法是#[item_attribute]（少了!）。

可以使用以下语法来给属性添加参数：

  • #[attribute = "value"]
  • #[attribute(key = "value")]
  • #[attribute(value)]

11.1 箱子

crate_type属性可被用来告诉编译器：这个箱子应被编译成一个可执行程序还是一个库（或者甚至指定是哪种类型的库）。crate_name属性则可以用来设置箱子的名字。

若使用了crate_type，就不必再向rustc添加--crate-type了。
