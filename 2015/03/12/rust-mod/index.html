Rust - 模块（Modules）

Mar 12 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

Rust 有一个强大的模块系统，它能够将代码按照层次划分成为一个个逻辑单元，同时控制这些单元之间的可见性（public/private）。

一个模块可以包含任何东西：函数、结构体（structs）、特征（traits）、实现块（imp block）或者其它模块等等。

9.1 可见性

模块中的条目默认皆为私有，你可以使用pub修饰符来使之对外公开可见。模块中只有公开的条目可以被此模块之外的代码调用。

9.2 use声明

可以使用use来将全称路径绑定到一个新的名字以方便调用。

9.3 super与self

可将super和self关键字用在调用路径中来避免歧义。
