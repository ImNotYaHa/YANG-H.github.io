Rust - 类型

Mar 11 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

Rust 通过静态类型推导来保证类型安全。你可以在变量声明的时候就标注出它的类型，不过在大多数情况下，编译器能够根据上下文自动推导变量的类型，从而免去人工标注的麻烦。

下面是 Rust 提供的基本类型：

  • 有符号整数：i8,i16,i32,i64,以及isize(指针大小)
  • 无符号整数：u8,u16,u32,u64,以及usize(指针大小)
  • 浮点数：f32,f64
  • char 即 Unicode 字符，如'a','α','∞'(占四个字节)
  • bool 即true或false
  • 以及 unit 类型 ()，此类型只有唯一的值 ()

类型转换

Rust 没有提供基本类型之间的隐式类型转换，但你可以使用as关键字来完成显式类型转换。

字面常量

数字字面常量可以通过添加类型后缀来标注类型，usize需要添加usize后缀，isize需要添加isize后缀。

无后缀的数字字面常量则需要其使用方式来决定类型。如果没有有效的约束，编译器默认将整数视为i32，将浮点数视为f64。

别名

type语句可以用来给一个已知类型赋一个新的名字。类型的名字必须满足CamelCase写法，否则编译器会产生警告。当然，基本类型除外，如usize，f32等等。

别名主要被用来节省代码量，譬如类型IoResult<T>就只是类型Result<T, IoError>的别名。
