Rust - 变量绑定

Mar 11 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

值（如字面常量）可以绑定到变量，这需要使用let来完成。

4.1 可变性

变量默认都是不可变的，通过添加 mut 修饰符以使之可变。

对于可变性相关的错误，编译器将抛出详细的诊断分析。

4.2 作用域与变量隐藏

每个变量都有作用域，变量被限制必须生存在某个 block 中。block 是使用大括号{}包含起来的一系列语句（statement）。Rust 同样支持变量隐藏。

4.3 前置声明

你可以把变量绑定的声明放在前面，之后再初始化它们。这不太常用，因为很容易导致使用未初始化的变量。

编译器禁止使用未初始化的绑定，因为这会产生未定义行为。
