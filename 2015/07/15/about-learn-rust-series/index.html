关于翻译 Rust by Example 系列

Jul 15 2015
  • 技术
  • Rust
  • 学习 Rust

Rust by Example 系列教程 之前翻译了 13 篇，然而一方面由于我自身学业事务繁杂，长时间没有跟进；另一方面 rust 在上半年发布了正式版（目前已经 1.2 beta1.9 了），期间语法变更不少，教程结构也变动巨大，按照之前的版本继续进行翻译缺乏实际价值。

因此我目前的想法是：不再严格按照官方结构和顺序进行翻译。未来有时间再针对一些相对 stable 的 feature 进行介绍。

然而事情的真相是：鉴于 rust 泛型至今不支持 numeric type parameter，我已经对 rust 无爱了，还是回归 C++ 怀抱吧！（;(）
