Rust - 属性（Attributes）

Mar 12 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

属性是用于描述模块、箱子或其它项目的元数据。属性可被用于：

  • 代码的条件编译
  • 设置箱子的名字、版本以及类别（是可执行程序还是库）
  • 禁用特定警告
  • 启用某些编译器特性（宏、全局导出(global import)等等）
  • 链接到非 Rust 库（foreign library）
  • 将函数标记为单元测试
  • 将函数标记为 benchmark 的一部分

Rust - 箱子（Crates）

Mar 12 2015
  • 技术
  • Rust
  • 学习 Rust

原文在此

一个箱子（crate）代表着 Rust 中的一个编译单元。只要执行rustc some_file.rs，some_file.rs就会被视作一个箱子文件（crate file）。如果some_file.rs中有声明任何mod，那么相关的模块成员都会在编译器编译之前合并到这个箱子文件中去。也就是说，模块不是被单独编译的，只有箱子文件可以被编译。
