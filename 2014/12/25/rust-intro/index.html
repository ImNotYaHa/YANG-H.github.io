Rust - 简介

Dec 25 2014
  • 技术
  • Rust
  • 学习 Rust

在这两年兴起的新语言中，Rust、Go 和 Swift 也许是最有名的三个，Rust 不算最新的，但就发布而言绝对是最晚的。

Rust 的设计目标是成为一种系统级别的语言，速度快，安全，无数据竞争，大有替代 C++ 的气势（鼓掌！！！）。

Rust 现在处于接近发布 1.0 版本的状态，虽然部分功能尚未完成，但整体语言性质已大体定型，我觉得是时候下手拓逃展避技现能实了！

Rust 的安装方法就不讲了，官网有安装包可直接用，教程也比较详细。
我的计划是首先翻译官方指南 Rust by Example（本想边学边写个小教程，但是稍微尝试便发现自己暂时不是原创的料;(），翻译完这个再考虑加些私货。

已经有大神们完成了部分翻译工作，譬如 Rust 中文开源社区。我自己的翻译可能会借鉴其中一些，先表示感谢~

下面是 Rust by Example 的介绍部分。

Rust 示例教程

Rust 是一个现代的系统级别编程语言，注重安全与速度。它使用无垃圾回收的内存安全机制来达到这两个目标。

Rust 示例教程包含多个示例，通过这些示例来介绍 Rust 中涉及到的各种概念以及标准库的用法。你可以从这里 check out 官方教程的源代码。

首先请确保你已安装 Rust、而且手边有文档，现在让我们开始吧！
